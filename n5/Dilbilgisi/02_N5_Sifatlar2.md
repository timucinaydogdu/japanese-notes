# 🗂️ な-Sıfatlar (な形容詞 / na-keiyoushi)

> **Seviye:** N5  
> **Konu:** な-sıfat çekimleri  
> **Son güncelleme:** 2026

---

## な-Sıfat nedir?

い-sıfatların aksine **kendi başına çekilmez** — だ/です yardımcısıyla çekilir. İsim önünde **な** eki alır, adını buradan alır. Görünüşte isme benzerler ama cümlede sıfat gibi davranırlar.

```
Türkçe:  "sessiz"       → çekilmez
          "sessiz değil" → ayrı kelime
          "sessizdi"     → ayrı yardımcı

Japonca: 静かだ          → sessiz       (だ ile çekildi)
          静かじゃない    → sessiz değil (じゃない eklendi)
          静かだった      → sessizdi     (だった eklendi)
```

> 💡 **Temel fark:** い-sıfat kendi çekilir (く、くない、かった...). な-sıfat だ/です yardımına muhtaçtır (じゃない、だった...).

| Form | Kural | Örnek (静か) | Türkçe |
|---|---|---|---|
| Olumlu samimi | + **だ** | 静か**だ** | sessiz |
| Olumlu kibar | + **です** | 静か**です** | sessizdir |
| Olumsuz samimi | + **じゃない** | 静か**じゃない** | sessiz değil |
| Olumsuz kibar | + **じゃないです** | 静か**じゃないです** | sessiz değildir |
| Geçmiş samimi | + **だった** | 静か**だった** | sessizdi |
| Geçmiş kibar | + **でした** | 静か**でした** | sessizdi (kibar) |
| Geçmiş olumsuz | + **じゃなかった** | 静か**じゃなかった** | sessiz değildi |
| Zarf formu | + **に** | 静か**に** | sessizce |
| Hale gelmek | + **になる** | 静か**になる** | sessizleşmek |
| İsim önünde | + **な** + isim | 静か**な**部屋 | sessiz oda |

## Çekim Detayları

### 1. Olumsuz — じゃない

**Kural:** な-sıfat + じゃない

```
静か  → 静かじゃない   (sessiz değil)
便利  → 便利じゃない   (kullanışlı değil)
元気  → 元気じゃない   (iyi/sağlıklı değil)
```

> ⚠️ **Sık yapılan hata:** い-sıfat gibi くない eklemek.
> ```
> 静かくない    ❌ YANLIŞ (い-sıfat kuralı)
> 静かじゃない  ✅ DOĞRU
> ```

**Resmi olumsuz:** じゃないです yerine **ではありません** de kullanılabilir (daha resmi).

**Örnekler:**
```
この図書館は静かじゃないです。   → Bu kütüphane sessiz değil.
彼は元気じゃない。               → O iyi değil.
このコードは綺麗じゃないです。   → Bu kod temiz değil.
```

---

### 2. Geçmiş Zaman — だった / でした

**Kural:** な-sıfat + だった (samimi) / でした (kibar)

```
静か  → 静かだった    (sessizdi)
暇   → 暇だった      (boştum)
有名  → 有名でした    (ünlüydü — kibar)
```

> ⚠️ **Sık yapılan hata:** い-sıfat gibi かった eklemek.
> ```
> 静かかった   ❌ YANLIŞ
> 静かだった   ✅ DOĞRU
> ```

**Geçmiş olumsuz:** じゃなかった

```
静か → 静かじゃなかった  (sessiz değildi)
```

**Örnekler:**
```
昨日は暇だった。           → Dün boştum.
その公園は静かでした。     → O park sessizdi.
彼は親切じゃなかった。     → O nazik değildi.
```

---

### 3. Zarf Formu — に

**Kural:** な-sıfat + に  
**Ne zaman?** Fiilden önce, sıfatı zarfa dönüştürmek için.

```
静か  → 静かに   (sessizce)
丁寧  → 丁寧に   (özenle, dikkatle)
親切  → 親切に   (nazikçe)
上手  → 上手に   (ustaca)
```

> ⚠️ **い-sıfat vs な-sıfat — zarf formu farkı:**
> ```
> い-sıfat: 早い → 早く (く eklenir)
> な-sıfat: 静か → 静かに (に eklenir)
> ```

**Örnekler:**
```
静かにしてください。       → Sessiz olun lütfen.
丁寧に書いてください。     → Özenle yazın lütfen.
真剣に勉強しています。     → Ciddi şekilde çalışıyorum.
```

---

### 4. になる — Hale Gelmek

**Kural:** な-sıfat + になる

```
有名  → 有名になる   → ünlü olmak
上手  → 上手になる   → iyileşmek, ustalaşmak
静か  → 静かになる   → sessizleşmek
```

> ⚠️ **い-sıfat vs な-sıfat — になる farkı:**
> ```
> い-sıfat: 高い → 高くなる   (く + なる)
> な-sıfat: 上手 → 上手になる (に + なる)
> ```

**Örnekler:**
```
日本語が上手になりたい。   → Japoncam iyileşsin istiyorum.
有名になりたい。           → Ünlü olmak istiyorum.
```

---

### 5. İsim Önünde — な

**Kural:** な-sıfat + **な** + isim  
Bu yüzden "**な**形容詞" denir!

```
静か  + な + 部屋    = 静かな部屋   (sessiz oda)
有名  + な + 人      = 有名な人     (ünlü kişi)
便利  + な + アプリ  = 便利なアプリ (kullanışlı uygulama)
大切  + な + データ  = 大切なデータ (önemli veri)
```

> ⚠️ **い-sıfat vs な-sıfat — isim önünde farkı:**
> ```
> い-sıfat: 高い店  → な eklenmez, そのまま
> な-sıfat: 静かな店 → な eklenir
> ```

## ⭐ Tuzak Sıfatlar — Sonu い ama な-Sıfat!

Bu sıfatlar sonu い ile biter ama **な-sıfat** gibi çekilir. En sık hata yapılan grup!

| Sıfat | Okunuş | Anlam | Kanıt |
|---|---|---|---|
| 好き | suki | sevmek | 好き**な**食べ物 ✅ |
| 嫌い | kirai | sevmemek | 嫌い**な**食べ物 ✅ |
| きれい | kirei | güzel / temiz | きれい**な**花 ✅ |

> ⚠️ **En sık yapılan hata:** Bu sıfatları い-sıfat gibi çekmek.
> ```
> 好きくない     ❌ YANLIŞ
> 好きじゃない   ✅ DOĞRU
>
> きれいくない    ❌ YANLIŞ
> きれいじゃない  ✅ DOĞRU
>
> きれいい花     ❌ YANLIŞ
> きれいな花     ✅ DOĞRU
> ```

> 💡 **Akılda kalıcı:** "Ki-su-ki" → **き**れい・**す**き・**き**らい — Üçü de sonu い ama な-sıfat!

---

## 🔍 上手 vs 得意 — İnce Fark

İkisi de "iyi olmak" anlamında ama **kullanım bağlamı farklı:**

| Sıfat | Kullanım | Örnek |
|---|---|---|
| 上手 (jouzu) | **Başkasının** becerisini methederken | 日本語が上手ですね！→ Japoncan çok iyi! |
| 得意 (tokui) | **Kendi** güçlü yönünden bahsederken | 数学が得意です。→ Matematiğim iyi. |

> ⚠️ **Dikkat:** Kendi hakkında 上手 kullanmak Japoncada kibar sayılmaz — mütevazılık kültürü gereği 得意 tercih edilir.

## ⚡ い-Sıfat vs な-Sıfat — Tam Karşılaştırma

| Form | い-sıfat (高い) | な-sıfat (静か) |
|---|---|---|
| Olumlu samimi | 高**い** | 静か**だ** |
| Olumlu kibar | 高**いです** | 静か**です** |
| Olumsuz samimi | 高**くない** | 静か**じゃない** |
| Olumsuz kibar | 高**くないです** | 静か**じゃないです** |
| Geçmiş samimi | 高**かった** | 静か**だった** |
| Geçmiş kibar | 高**かったです** | 静か**でした** |
| Geçmiş olumsuz | 高**くなかった** | 静か**じゃなかった** |
| Zarf formu | 高**く** | 静か**に** |
| Hale gelmek | 高**くなる** | 静か**になる** |

> 💡 **Hızlı hatırlama:**

## 🧠 Pratik Cümle Örnekleri

```
この図書館は静かで、広いです。
→ Bu kütüphane sessiz ve geniş.
  (て形: 静か → 静かで → iki sıfatı bağlar)

昨日は暇だったから、映画を見ました。
→ Dün boştum, o yüzden film izledim.

有名なエンジニアになりたいです。
→ Ünlü bir mühendis olmak istiyorum.

静かにしてください。
→ Sessiz olun lütfen.

好きな食べ物は何ですか？
→ Sevdiğin yemek nedir?
  (好き → な-sıfat → 好きな + 食べ物)
```
