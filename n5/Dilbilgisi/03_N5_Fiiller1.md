# 🗂️ て形 (te-kei) — Te Formu

> **Seviye:** N5  
> **Konu:** Fiillerin て formu — bağlaç / sıralama  
> **Son güncelleme:** 2026

---

## て形 nedir?

Japoncada en çok kullanılan fiil formudur. Türkçedeki **"-ip, -arak, -erek"** eklerine benzer — eylemleri birbirine bağlar, sıralar. Ayrıca rica etme, izin, yasak, devam gibi pek çok yapının temelini oluşturur.

```
起きて、シャワーを浴びて、仕事に行きます。
→ Kalkıp, duş alıp, işe gidiyorum.

コードを書いて、テストしました。
→ Kod yazıp test ettim.
```

> 💡 **Önemli:** て形 cümleyi bitirmez, **bağlar**. Cümle sonu değil, köprüdür.

---

## Fiil Grupları

Japoncada 3 fiil grubu var. て形 yapmak için önce grubunu bilmen gerekir.

| Grup | Diğer adı | Kural zorluk |
|---|---|---|
| Grup 1 | u-fiilleri / 五段動詞 | En çok kural |
| Grup 2 | ru-fiilleri / 一段動詞 | En kolay |
| Grup 3 | Düzensiz | Sadece 2 fiil, ezber |

### Grubu nasıl anlarım?

```
1. Sonu る ile bitmiyor mu?
   → Kesinlikle Grup 1. Bitti.

2. Sonu る ile bitiyor mu?
   → る'dan önce い/え sesi mi?
       → EVET → Grup 2 (ama tuzak listesini kontrol et!)
       → HAYIR (あ/う/お sesi) → Grup 1
```

> ⚠️ **Tuzak fiiller:** Sonu いる/える gibi görünür ama Grup 1 olan fiiller:
> 帰る・走る・切る・知る・入る・要る → hepsi **Grup 1!**

---

## Grup 1 — て形 Kuralları

ます形'den ます'ı at, kalan **son sese** göre değiştir:

| Son ses (ます kökü) | → て形 | Örnek fiiller |
|---|---|---|
| **き** | → **いて** | 書く→書いて / 聞く→聞いて / 働く→働いて / 歩く→歩いて |
| **ぎ** | → **いで** | 泳ぐ→泳いで / 急ぐ→急いで / 脱ぐ→脱いで |
| **し** | → **して** | 話す→話して / 貸す→貸して / 押す→押して / 消す→消して / 返す→返して |
| **ち・り・い** | → **って** | 待つ→待って / 持つ→持って / 帰る→帰って / 走る→走って / 切る→切って / 買う→買って / 言う→言って / 使う→使って |
| **に・び・み** | → **んで** | 死ぬ→死んで / 遊ぶ→遊んで / 呼ぶ→呼んで / 飲む→飲んで / 読む→読んで / 住む→住んで |

> ⚠️ **行く tek istisna!**
> ```
> 行く → 行いて ❌ YANLIŞ (き → いて kuralına uymaz)
> 行く → 行って ✅ DOĞRU
> ```

> 💡 **Akılda kalıcı sıra:**
> き**いて** → ぎ**いで** → し**して** → ち/り/い**って** → に/び/み**んで**

## Grup 2 — て形 Kuralları

En kolay grup. **ます'ı at, て ekle.** Hepsi bu.

```
食べます → 食べ + て = 食べて
起きます → 起き + て = 起きて
見ます   → 見  + て = 見て
```

| Fiil | Anlam | て形 |
|---|---|---|
| 食べる | yemek | 食べて |
| 起きる | uyanmak | 起きて |
| 見る | görmek / izlemek | 見て |
| 寝る | uyumak | 寝て |
| 着る | giymek | 着て |
| 開ける | açmak | 開けて |
| 教える | öğretmek | 教えて |
| 借りる | ödünç almak | 借りて |
| 忘れる | unutmak | 忘れて |
| 覚える | ezberlemek | 覚えて |
| 考える | düşünmek | 考えて |
| 調べる | araştırmak | 調べて |
| 変える | değiştirmek | 変えて |
| 続ける | devam etmek | 続けて |
| 出かける | dışarı çıkmak | 出かけて |

## Grup 3 — Düzensiz

Sadece 2 fiil. Kural yok, direkt ezber.

| Fiil | て形 |
|---|---|
| する (yapmak) | **して** |
| くる (gelmek) | **きて** |

> 💡 **する bileşik fiiller** — する'lu her fiil aynı şekilde:
> ```
> 勉強する → 勉強して  (çalışmak)
> 電話する → 電話して  (telefon etmek)
> 運動する → 運動して  (egzersiz yapmak)
> コピーする → コピーして
> インストールする → インストールして
> ```

## て形'in Kullanım Alanları

### 1. Eylemleri Sıralama / Bağlama
**Türkçe karşılık:** "-ip, -arak, -erek"

```
起きて、シャワーを浴びて、朝ごはんを食べます。
→ Kalkıp, duş alıp, kahvaltı yapıyorum.

コードを書いて、テストしました。
→ Kod yazıp test ettim.
```

---

### 2. 〜てください — Rica Etme
**Türkçe karşılık:** "...yapın lütfen"

```
ちょっと待ってください。          → Biraz bekleyin lütfen.
ここに名前を書いてください。      → Buraya isminizi yazın lütfen.
静かにしてください。              → Sessiz olun lütfen.
```

---

### 3. 〜ている — Devam Eden Eylem
**Türkçe karşılık:** "-iyor" (şimdiki zaman)

> 💡 Bu yapı ayrı bir konu olarak ele alınmıştır → [[N5_Fiil_Te_Iru]]

```
今、コードを書いています。  → Şu an kod yazıyorum.
日本語を勉強しています。    → Japonca çalışıyorum.
```

---

### 4. 〜てもいい — İzin
**Türkçe karşılık:** "...yapabilir miyim"

```
ここで食べてもいいですか。  → Burada yiyebilir miyim?
帰ってもいいです。          → Eve gidebilirsiniz.
```

---

### 5. 〜てはいけない — Yasak
**Türkçe karşılık:** "...yapılamaz, yasak"

```
ここで写真を撮ってはいけません。  → Burada fotoğraf çekilemez.
タバコを吸ってはいけません。      → Sigara içilemez.
```

---

### 6. 〜てから — ...Yaptıktan Sonra
**Türkçe karşılık:** "...yaptıktan sonra"

```
ご飯を食べてから、勉強します。  → Yemek yedikten sonra çalışacağım.
考えてから、返事します。         → Düşündükten sonra cevap vereceğim.
```

> ⚠️ **〜てから vs 前に karıştırma:**
> ```
> 食べてから → yedikten SONRA
> 食べる前に → yemekten ÖNCE
> ```
> → Detay için [[N5_Fiil_Mae_ni]]

## Sıfatlarda て形 — Bağlaç

### い-sıfat → くて

**Kural:** い → くて

```
このレストランは安くて、美味しいです。
→ Bu restoran ucuz ve lezzetli.
```

### な-sıfat → で

**Kural:** な-sıfat + で

```
彼は親切で、頭がいい。
→ O nazik ve zeki.
```

> ⚠️ **いい → よくて (istisna!)**
> ```
> いいくて ❌ YANLIŞ
> よくて   ✅ DOĞRU
> ```

---

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 行いて | 行って | 行く tek istisna |
| 飲みて | 飲んで | み → んで |
| 食べって | 食べて | Grup 2: sadece て |
| 帰りて | 帰って | 帰る Grup 1 tuzak (り→って) |
| 走りて | 走って | 走る Grup 1 tuzak |
| 高いくて | 高くて | い-sıfat て形: い → くて |
| 静かくて | 静かで | な-sıfat て形: + で |

## 🧠 Pratik Cümle Örnekleri

```
朝起きて、コーヒーを飲んで、仕事をします。
→ Sabah kalkıp, kahve içip, çalışıyorum.

このライブラリは便利で、使いやすいです。
→ Bu kütüphane kullanışlı ve kullanımı kolay.

インストールしてから、設定してください。
→ Kurduktan sonra ayarlayın lütfen.

ここでタバコを吸ってはいけません。
→ Burada sigara içilemez.

もう少し考えてから、返事します。
→ Biraz daha düşündükten sonra cevap vereceğim.
```
