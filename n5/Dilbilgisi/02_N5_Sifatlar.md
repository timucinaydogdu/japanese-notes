# 🗂️ い-Sıfatlar (い形容詞 / i-keiyoushi)

> **Seviye:** N5  
> **Konu:** い-sıfat çekimleri  
> **Son güncelleme:** 2026

---

## い-Sıfat nedir?

Sonu **い** ile biten sıfatlardır. Türkçedeki sıfatlardan en büyük farkı: **kendi başına çekilir**, yani yardımcı fiile gerek yoktur. Tıpkı bir fiil gibi zaman ve olumsuzluk alır.

```
Türkçe:  "pahalı"       → çekilmez
          "pahalı değil" → ayrı kelime gerekir
          "pahalıydı"    → ayrı yardımcı fiil gerekir

Japonca: 高い            → pahalı
          高くない        → pahalı değil   (kendi çekildi!)
          高かった        → pahalıydı      (kendi çekildi!)
```

> 💡 **Temel kural:** Tüm çekimler **い'yi değiştirerek** yapılır. い'yi at, yeni ek getir.

## Çekim Tablosu

| Form | Kural | Örnek (高い) | Türkçe |
|---|---|---|---|
| Olumlu samimi | そのまま | 高**い** | pahalı |
| Olumlu kibar | + です | 高**いです** | pahalıdır |
| Olumsuz samimi | い → **くない** | 高**くない** | pahalı değil |
| Olumsuz kibar | い → **くないです** | 高**くないです** | pahalı değildir |
| Geçmiş samimi | い → **かった** | 高**かった** | pahalıydı |
| Geçmiş kibar | い → **かったです** | 高**かったです** | pahalıydı (kibar) |
| Geçmiş olumsuz | い → **くなかった** | 高**くなかった** | pahalı değildi |
| Zarf formu | い → **く** | 高**く** | pahalıca / yüksekçe |
| Hale gelmek | い → **くなる** | 高**くなる** | pahalılaşmak |
| İsim önünde | değişmez | 高**い**店 | pahalı dükkan |

## Çekim Detayları

### 1. Olumsuz — くない

**Kural:** い → く yap, sonra ない ekle.

```
高い → 高く + ない = 高くない
寒い → 寒く + ない = 寒くない
難しい → 難しく + ない = 難しくない
```

> ⚠️ **Sık yapılan hata:** い'yi silmeden ない eklemek.
> ```
> 高いない ❌ YANLIŞ
> 高くない ✅ DOĞRU
> ```

**Örnekler:**
```
このカフェは高くないです。     → Bu kafe pahalı değil.
今日は寒くない。               → Bugün soğuk değil.
このコードは難しくないです。   → Bu kod zor değil.
```

---

### 2. Geçmiş Zaman — かった

**Kural:** い → かった

```
高い  → 高かった    (pahalıydı)
寒い  → 寒かった    (soğuktu)
楽しい → 楽しかった (eğlenceliydi)
```

> 💡 **Akılda kalıcı:** "かった" = "kattı" → geçmişe kattı!

**Geçmiş olumsuz:** い → くなかった

```
高い → 高くなかった   (pahalı değildi)
```

**Örnekler:**
```
昨日は寒かった。                       → Dün soğuktu.
このレストランは高くなかったです。     → Bu restoran pahalı değildi.
テストは難しかったです。               → Sınav zordu.
```

---

### 3. Zarf Formu — く

**Kural:** い → く  
**Ne zaman?** Fiilden önce, sıfatı zarfa dönüştürmek için.

```
早い → 早く   (hızlıca / erken)
大きい → 大きく (büyük şekilde)
上手い → 上手く (ustaca)
```

**Örnekler:**
```
早く来てください。           → Erken/hızlıca gelin.
安く売っています。           → Ucuza satıyor.
```

---

### 4. くなる — Hale Gelmek

**Kural:** い → く + なる  
**Ne anlama gelir?** "...hale gelmek, ...olmak"

```
高い  → 高くなる    → pahalılaşmak
上手い → 上手くなる  → iyileşmek, ustalaşmak
大きい → 大きくなる  → büyümek
```

**Örnekler:**
```
日本語が上手くなりたい。  → Japoncam iyileşsin istiyorum.
天気が寒くなった。        → Hava soğudu (soğuklaştı).
```

---

### 5. İsim Önünde Kullanım

い-sıfat isimden önce **hiçbir değişiklik olmadan** gelir.

```
高い + レストラン = 高いレストラン   (pahalı restoran)
難しい + 問題    = 難しい問題       (zor problem)
新しい + パソコン = 新しいパソコン  (yeni bilgisayar)
```

> 💡 **Olumsuz sıfat da isim önüne gelebilir:**
> ```
> 高くないレストラン → Pahalı olmayan restoran
> ```

## ⭐ いい — Tek İstisna

**いい (iyi)** tek düzensiz い-sıfatıdır. Çekilirken **よ** köküne döner.

| Form | いい | Türkçe |
|---|---|---|
| Olumlu | **いい** / よい | iyi |
| Olumsuz | **よく**ない | iyi değil |
| Geçmiş | **よ**かった | iyiydi |
| Geçmiş olumsuz | **よく**なかった | iyi değildi |
| Zarf | **よく** | iyi şekilde / sık |
| Hale gelmek | **よく**なる | iyileşmek |

> ⚠️ **En sık yapılan hata:** いい'yi düzenli gibi çekmek.
> ```
> いくない   ❌ YANLIŞ
> よくない   ✅ DOĞRU
>
> いかった   ❌ YANLIŞ
> よかった   ✅ DOĞRU
> ```

> 💡 **よかった！** — Japoncada en çok duyacağın ifadelerden biri. "Ne iyi!, İyi ki!" anlamında, haber alınca veya rahatlanınca kullanılır.

**Örnekler:**
```
成功してよかった！       → Başarılı olduğun için ne iyi!
よくわかりません。       → Pek anlamıyorum.
天気がよくないです。     → Hava iyi değil.
日本語がよくなりたい。   → Japoncam iyileşsin istiyorum.
```

## 🌡️ Önemli Sıfat Çiftleri

### 暑い (atsui) vs 熱い (atsui)
İkisi de **aynı şekilde okunur** ama farklı anlam:

| Sıfat | Anlam | Kullanım |
|---|---|---|
| 暑い | sıcak | **hava** sıcak (dışarıda) |
| 熱い | sıcak | **nesne** sıcak (çay, yemek, bilgisayar...) |

```
今日は暑い。     → Bugün hava sıcak.
このお茶は熱い。 → Bu çay sıcak.
```

> ⚠️ **Programcı notu:** CPUが熱い → İşlemci sıcak! (熱い kullanılır)

### 寒い (samui) vs 冷たい (tsumetai)

| Sıfat | Anlam | Kullanım |
|---|---|---|
| 寒い | soğuk | **hava** soğuk |
| 冷たい | soğuk | **nesne/içecek** soğuk |

```
今日は寒い。      → Bugün soğuk.
この水は冷たい。  → Bu su soğuk.
```

### 易しい (yasashii) vs 優しい (yasashii)
İkisi de **aynı okunuş**, farklı anlam — dikkat!

| Sıfat | Anlam |
|---|---|
| 易しい | kolay |
| 優しい | nazik, şefkatli |

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 高いない | 高くない | い → く, sonra ない |
| 高いだった | 高かった | な-sıfat gibi çekilmez |
| いくない | よくない | いい istisnası |
| いかった | よかった | いい istisnası |
| 難しいです (olumsuz) | 難しくないです | olumsuz için くない |

## 🧠 Pratik Cümle Örnekleri

```
このレストランは高くて、美味しいです。
→ Bu restoran pahalı ama lezzetli.
  (て形: 高い → 高くて → iki sıfatı bağlar)

昨日のテストは難しかったです。
→ Dünkü sınav zordu.

日本語が上手くなりたいです。
→ Japoncam iyileşsin istiyorum.

早く来てください。
→ Çabuk gelin lütfen.

よかった！試験に合格しました！
→ Ne iyi! Sınavı geçtim!
```
