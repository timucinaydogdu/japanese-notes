# 🗂️ Var / Yok (存在表現)

> **Seviye:** N5  
> **Konular:** があります / がいます  
> **Son güncelleme:** 2026

---

## Genel Bakış

Japoncada "var/yok" ifadesi için **iki farklı fiil** kullanılır. Hangisini kullanacağını belirleyen tek şey: **varlığın canlı mı, cansız mı olduğu.**

| Fiil | Ne için? | Türkçe |
|---|---|---|
| **あります** | Cansız varlıklar | var (nesne, bitki, bina...) |
| **います** | Canlı varlıklar | var (insan, hayvan...) |

> ⚠️ **Bu ayrım Türkçede yok** — ikisi de "var" demek ama Japoncada kesinlikle karıştırılamaz.

---

## 1. があります — Cansız Varlıklar

### Ne yapar?
**Cansız şeylerin** var olduğunu ifade eder. Nesneler, bitkiler, binalar, araçlar, olaylar, zaman...

### Yapı
```
[Yer] に [Nesne] が あります
```

```
机の上に本があります。         → Masanın üstünde kitap var.
駅の前にコンビニがあります。   → İstasyonun önünde market var.
冷蔵庫にジュースがあります。   → Buzdolabında meyve suyu var.
明日、会議があります。         → Yarın toplantı var.
```

### Çekim Tablosu

| Form | あります | Türkçe |
|---|---|---|
| Olumlu kibar | **あります** | var |
| Olumsuz kibar | **ありません** | yok |
| Geçmiş kibar | **ありました** | vardı |
| Geçmiş olumsuz | **ありませんでした** | yoktu |
| Olumlu samimi | **ある** | var |
| Olumsuz samimi | **ない** | yok |

```
お金がありません。             → Param yok.
昨日、授業がありました。       → Dün ders vardı.
時間がありませんでした。       → Zaman yoktu.
```

---

## 2. がいます — Canlı Varlıklar

### Ne yapar?
**Canlı varlıkların** var olduğunu ifade eder. İnsanlar, hayvanlar, böcekler...

### Yapı
```
[Yer] に [Canlı] が います
```

```
部屋に猫がいます。             → Odada kedi var.
公園に子供がいます。           → Parkta çocuk var.
会社に社長がいます。           → Şirkette müdür var.
庭に犬がいます。               → Bahçede köpek var.
```

### Çekim Tablosu

| Form | います | Türkçe |
|---|---|---|
| Olumlu kibar | **います** | var |
| Olumsuz kibar | **いません** | yok |
| Geçmiş kibar | **いました** | vardı |
| Geçmiş olumsuz | **いませんでした** | yoktu |
| Olumlu samimi | **いる** | var |
| Olumsuz samimi | **いない** | yok |

```
今、誰もいません。             → Şu an kimse yok.
昨日、彼女がいました。         → Dün o (kız) vardaydı.
子供がいますか？               → Çocuğunuz var mı?
```

---

## あります vs います — Karar Tablosu

| Varlık | Fiil | Örnek |
|---|---|---|
| Kitap, kalem, masa | あります | 本**があります** |
| Araba, bina, park | あります | 車**があります** |
| Para, zaman, iş | あります | お金**があります** |
| Bitki, ağaç, çiçek | あります | 木**があります** |
| İnsan, çocuk, arkadaş | います | 友達**がいます** |
| Köpek, kedi, kuş | います | 犬**がいます** |
| Böcek, balık | います | 魚**がいます** |

> ⚠️ **Bitkiler あります alır** — canlı olsalar da います değil.
> ```
> 庭に木があります。  ✅ (Bahçede ağaç var)
> 庭に木がいます。   ❌ YANLIŞ
> ```

> ⚠️ **Robot, oyuncak, karikatür karakteri?**  
> Gerçek hayatta hareket edemez → あります  
> Ama anime/manga bağlamında "yaşayan" gibi davranıyorsa → います (bağlama göre değişir)

---

## に Partikülü ile Kullanım

あります ve います çoğunlukla **に** partikülü ile yer belirtir:

```
[Yer] に [Varlık] が あります/います
```

```
駅の前にATMがあります。        → İstasyonun önünde ATM var.
教室に学生がいます。           → Sınıfta öğrenci var.
引き出しの中にカギがあります。 → Çekmecenin içinde anahtar var.
```

> 💡 **Soru sorarken:**
> ```
> トイレはどこにありますか？     → Tuvalet nerede?
> 田中さんはどこにいますか？     → Tanaka nerede?
> ```

---

## Sahiplik İfadesi

あります ve います **sahip olmak** anlamında da kullanılır:

```
兄弟がいますか？               → Kardeşiniz var mı?
はい、妹がいます。             → Evet, kız kardeşim var.
いいえ、いません。             → Hayır, yok.

車がありますか？               → Arabanız var mı?
はい、あります。               → Evet, var.
```

> 💡 **Sahiplik için も kullanımı:**
> ```
> 猫もいます。    → Kedi de var.
> お金もありません。→ Para da yok.
> ```

---

## ある / いる — Samimi Form

Günlük konuşmada **ある/いる** sıkça kullanılır:

```
あそこに誰かいる。             → Orada biri var.
お金ある？                     → Para var mı? (çok samimi)
時間ない。                     → Zaman yok.
彼女いるの？                   → Kız arkadaşın var mı?
```

---

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 猫があります | 猫がいます | Kedi canlı → います |
| 本がいます | 本があります | Kitap cansız → あります |
| 木がいます | 木があります | Bitki cansız → あります |
| 友達があります | 友達がいます | İnsan canlı → います |
| 魚があります | 魚がいます | Balık canlı → います |
| いますか (nesne için) | ありますか | Nesne için あります |

## 🧠 Pratik Cümle Örnekleri

```
この近くにスーパーがありますか？
→ Bu yakında market var mı?

駅の近くに大きい公園があります。
→ İstasyonun yakınında büyük park var.

今日、授業がありません。
→ Bugün ders yok.

部屋に誰かいますか？
→ Odada biri var mı?

子供が二人います。
→ İki çocuğum var.

冷蔵庫に何もありません。
→ Buzdolabında hiçbir şey yok.

昨日、ここに誰もいませんでした。
→ Dün burada kimse yoktu.
```
