# 🗂️ 〜前に (mae ni) — ...Den Önce

> **Seviye:** N5  
> **Konu:** 前に yapısı — zaman sırası  
> **Son güncelleme:** 2026

---

## 〜前に nedir?

**Bir eylemden veya zamandan önce** gerçekleşen şeyi anlatır. Türkçedeki **"...den önce, ...madan önce"** karşılığıdır.

```
寝る前に、歯を磨きます。
→ Uyumadan önce dişlerimi fırçalarım.

食事の前に、手を洗います。
→ Yemekten önce ellerimi yıkarım.
```

---

## Yapı

### Fiil + 前に

> **Kural:** Fiil her zaman **辞書形 (sözlük formu)** ile gelir — geçmiş veya te formu değil!

```
[Fiil sözlük formu] + 前に + [ana eylem]
```

| Örnek | Türkçe |
|---|---|
| 寝る**前に** | uyumadan önce |
| 食べる**前に** | yemeden önce |
| 行く**前に** | gitmeden önce |
| 始める**前に** | başlamadan önce |
| 使う**前に** | kullanmadan önce |

```
寝る前に、本を読みます。      → Uyumadan önce kitap okurum.
食べる前に、手を洗ってください。→ Yemeden önce ellerini yıkayın.
会議の前に、資料を確認します。 → Toplantıdan önce belgeleri kontrol ederim.
```

---

### İsim + の + 前に

İsimle kullanıldığında araya **の** girer:

```
[İsim] + の + 前に + [ana eylem]
```

| Örnek | Türkçe |
|---|---|
| 食事**の前に** | yemekten önce |
| 授業**の前に** | dersten önce |
| 試験**の前に** | sınavdan önce |
| 寝る**前に** | uyumadan önce (fiil) |

```
授業の前に、単語を覚えます。   → Dersten önce kelimeleri ezberlerim.
試験の前に、よく寝てください。 → Sınavdan önce iyi uyuyun.
仕事の前に、コーヒーを飲みます。→ İşten önce kahve içerim.
```

---

## ⚠️ En Önemli Kural — Sözlük Formu!

前に ile gelen fiil **geçmişte olsa bile sözlük formu** kalır. Türkçe mantığıyla düşününce bu tuhaf gelebilir.

> ⚠️ **Sık yapılan hata:** た形 (geçmiş) ile kullanmak.
> ```
> 寝た前に     ❌ YANLIŞ
> 寝る前に     ✅ DOĞRU
>
> 食べた前に   ❌ YANLIŞ
> 食べる前に   ✅ DOĞRU
> ```

> 💡 **Neden sözlük formu?** Çünkü 前に "o eylem henüz gerçekleşmeden önce" demek. Eylem henüz olmadığı için geçmiş form kullanılmaz.

## 前に vs 〜てから — Karıştırma

Bu iki yapı sıkça birbirine karışır:

| Yapı | Anlam | Örnek |
|---|---|---|
| 〜前に | ...den ÖNCE | 食べる**前に**手を洗う → Yemeden ÖNCE el yıka |
| 〜てから | ...yaptıktan SONRA | 食べ**てから**歯を磨く → Yedikten SONRA diş fırçala |

```
コードを書く前に、設計を考えます。
→ Kod yazmadan ÖNCE tasarımı düşünürüm.

コードを書いてから、テストします。
→ Kod yazdıktan SONRA test ederim.
```

> 💡 **Akılda kalıcı:**
> - 前 = "ön, ileri" → önce
> - てから = "yaptıktan itibaren" → sonra

---

## 前に vs 後で — Zaman Çifti

前に'nin tam zıttı **後で (ato de)** — "...den sonra":

| Yapı | Anlam | Örnek |
|---|---|---|
| 〜前に | ...den önce | 寝る**前に** → uyumadan önce |
| 〜後で | ...den sonra | 寝た**後で** → uyuduktan sonra |

> ⚠️ **後で dikkat:** 後で fiil **た形** (geçmiş) ile gelir — 前に'nin tam tersi!
> ```
> 食べる前に  → yemeden önce   (sözlük formu)
> 食べた後で  → yedikten sonra  (た形)
> ```

---

## Cümle İçindeki Konumu

前に cümlesi **ana eylemden önce** gelir. Türkçeyle aynı sıra:

```
[Önce yapılan şey] 前に、[ana eylem]

寝る前に、日本語を勉強します。
→ Uyumadan önce Japonca çalışırım.
↑ önce olan          ↑ ana eylem
```

---

## Zaman İfadeleriyle 前に

前に tek başına zaman zarfı olarak da kullanılır — "daha önce, eskiden":

```
前に日本に行ったことがあります。
→ Daha önce Japonya'ya gitmiştim.

前に会いましたね。
→ Daha önce tanışmıştık, değil mi?
```

> 💡 Bu kullanımda 前 tek başına gelir, fiil veya isim almaz.

---

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 寝た前に | 寝る前に | 前に sözlük formu ister |
| 食べて前に | 食べる前に | て形 değil, sözlük formu |
| 授業前に (isim) | 授業の前に | İsim + の + 前に |
| 前に寝る (sıralama) | 寝る前に | 前に ana eylemden önce gelir |

---

## 🧠 Pratik Cümle Örnekleri

```
日本に来る前に、日本語を少し勉強しました。
→ Japonya'ya gelmeden önce biraz Japonca çalıştım.

プログラムを実行する前に、バックアップを取ってください。
→ Programı çalıştırmadan önce yedek alın lütfen。

寝る前に、スマホを見ないほうがいいですよ。
→ Uyumadan önce telefona bakmamalısın.

試験の前に、よく寝ることが大切です。
→ Sınavdan önce iyi uyumak önemli.

会議の前に、このレポートを読んでおいてください。
→ Toplantıdan önce bu raporu okuyun.
```
