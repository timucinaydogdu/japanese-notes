# 🗂️ Rica / Yasak (依頼・禁止表現)

> **Seviye:** N5  
> **Konular:** 〜てください / ないでください / 〜といけない / ちゃいけない・じゃいけない  
> **Son güncelleme:** 2026

---

## Genel Bakış

Bu 4 yapı **rica ve yasak** ifadelerinin temelini oluşturur. Tonu ve resmiyet derecesi farklı:

| Yapı | Anlam | Ton |
|---|---|---|
| 〜てください | ...yapın lütfen | Kibar rica |
| 〜ないでください | ...yapmayın lütfen | Kibar olumsuz rica |
| 〜てはいけない | ...yapılmaz / yasak | Kural / yasak |
| ちゃいけない・じゃいけない | ...yapılmaz (kısaltma) | Günlük / samimi yasak |

---

## 1. 〜てください — Yapın Lütfen

### Ne yapar?
Karşındakinden bir şey yapmasını **kibarca ister**. Türkçedeki **"...yapın lütfen, ...yapar mısınız"** karşılığıdır.

### Yapı
> **て形 + ください**

```
書く  → 書いて + ください = 書いてください    (yazın lütfen)
待つ  → 待って + ください = 待ってください    (bekleyin lütfen)
見る  → 見て   + ください = 見てください      (bakın lütfen)
来る  → きて   + ください = きてください      (gelin lütfen)
```

```
ちょっと待ってください。           → Biraz bekleyin lütfen.
ここに名前を書いてください。       → Buraya isminizi yazın lütfen.
静かにしてください。               → Sessiz olun lütfen.
もう一度言ってください。           → Bir kez daha söyleyin lütfen.
ドアを開けてください。             → Kapıyı açın lütfen.
```

### Daha Kibar Form — てくださいませんか

Rica'yı daha da kibarca yapmak için **てくださいませんか** kullanılır:

```
てください        → ...yapın lütfen       (standart)
てくださいませんか → ...yapar mısınız?    (daha kibar)
```

```
もう一度説明してくださいませんか。
→ Bir kez daha açıklar mısınız?
```

> 💡 **Günlük konuşmada** ください düşebilir:
> ```
> 待ってください → 待って！  (bekle! — samimi)
> 見てください  → 見て！    (bak! — samimi)
> ```

## 2. 〜ないでください — Yapmayın Lütfen

### Ne yapar?
Karşındakinden bir şey **yapmamasını** kibarca ister. てください'nin olumsuz karşılığıdır.

### Yapı
> **ない形 + でください**

> ⚠️ **Dikkat:** て形 değil, **ない形** kullanılır!

```
書く  → 書かない + でください = 書かないでください    (yazmayın)
待つ  → 待たない + でください = 待たないでください    (beklemeyin)
食べる → 食べない + でください = 食べないでください   (yemeyin)
する  → しない   + でください = しないでください      (yapmayın)
```

```
ここで写真を撮らないでください。   → Burada fotoğraf çekmeyin lütfen.
タバコを吸わないでください。       → Sigara içmeyin lütfen.
忘れないでください。               → Unutmayın lütfen.
遅れないでください。               → Geç kalmayın lütfen.
心配しないでください。             → Endişelenmeyin lütfen.
```

> ⚠️ **Sık yapılan hata:** て形 ile olumsuz yapmaya çalışmak.
> ```
> 食べてくないでください ❌ YANLIŞ
> 食べないでください    ✅ DOĞRU
> ```

---

## 3. 〜てはいけない — Yasak / Yapılmaz

### Ne yapar?
Bir şeyin **yapılmasının yasak olduğunu** ifade eder. ないでください kişisel bir ricayken, てはいけない bir **kural veya genel yasak** bildirir. Türkçedeki **"...yapılamaz, ...yapılmaz, ...yasaktır"** karşılığıdır.

### Yapı
> **て形 + はいけない / はいけません**

```
撮る  → 撮って + はいけない = 撮ってはいけない    (çekilemez)
吸う  → 吸って + はいけない = 吸ってはいけない    (içilemez)
入る  → 入って + はいけない = 入ってはいけない    (girilemez)
```

```
ここで写真を撮ってはいけません。   → Burada fotoğraf çekilemez.
授業中に携帯を使ってはいけません。 → Ders sırasında telefon kullanılamaz.
この部屋に入ってはいけません。     → Bu odaya girilemez.
嘘をついてはいけません。           → Yalan söylenmez.
```

### ないでください vs てはいけない — Fark

> ⚠️ **Bu iki yapı çok karıştırılıyor:**

| | ないでください | てはいけない |
|---|---|---|
| Anlam | Rica / istek | Yasak / kural |
| Kim söyler? | Kişisel istek | Genel kural, otorite |
| Ton | Nazik, kişisel | Kesin, resmi |
| Örnek | タバコを吸わないでください (sigarayı bırakmanızı rica ederim) | タバコを吸ってはいけません (sigara içmek yasaktır) |

> 💡 **Kısa kural:**
> - ないでください = "Yapma lütfen" (kişisel)
> - てはいけない = "Yapılmaz!" (kural)

## 4. ちゃいけない / じゃいけない — Günlük Yasak

### Ne yapar?
てはいけない'nin **konuşma dili kısaltmasıdır**. Anlam aynı, sadece daha hızlı ve samimi.

### Kısaltma Kuralı

> **てはいけない → ちゃいけない**  
> **でははいけない → じゃいけない**

```
撮ってはいけない → 撮っちゃいけない
食べてはいけない → 食べちゃいけない
飲んではいけない → 飲んじゃいけない
```

```
そんなこと言っちゃいけないよ。   → Öyle şey söylenmez.
ここで遊んじゃいけないよ。       → Burada oynanmaz.
嘘ついちゃいけない。             → Yalan söylenmez.
それ食べちゃいけないよ。         → Onu yememelisin.
```

> 💡 **Ne zaman kullanılır?**
> - Arkadaşlarla konuşurken
> - Ebeveyn çocuğuna söylerken
> - Samimi uyarılarda
> - Sınavda **kullanma** — resmi değil

### Resmi vs Samimi Karşılaştırma

| Resmi | Samimi | Türkçe |
|---|---|---|
| 食べてはいけません | 食べちゃいけない | yemek yasak |
| 飲んではいけません | 飲んじゃいけない | içmek yasak |
| 入ってはいけません | 入っちゃいけない | giremezsin |
| 使ってはいけません | 使っちゃいけない | kullanamazsın |

---

## 4 Yapının Karşılaştırması

```
タバコを吸わないでください。    → Sigara içmeyin lütfen. (kişisel rica)
タバコを吸ってはいけません。    → Sigara içmek yasaktır. (kural)
タバコ吸っちゃいけないよ。      → Sigara içilmez ha. (samimi uyarı)
タバコを吸ってください。         → Sigara için lütfen. (←nadiren bu)
```

## ない形 Hatırlatması

ないでください için ない形 bilmen gerekir. Hızlı özet:

| Grup | Kural | Örnek |
|---|---|---|
| Grup 1 | う → あない | 書く→書かない / 待つ→待たない / 飲む→飲まない / 買う→買わない |
| Grup 2 | る → ない | 食べる→食べない / 見る→見ない |
| Grup 3 | する→しない / くる→こない | — |

> ⚠️ **買う dikkat:** 買う → 買わない (買あない değil!)

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 食べてくないでください | 食べないでください | ないでください → ない形 + でください |
| 書かないてください | 書かないでください | で gerekli, て değil |
| 食べるはいけない | 食べてはいけない | て形 + はいけない |
| ちゃいけない (sınavda) | てはいけません | Sınavda resmi form kullan |
| 待ってはいけない (rica) | 待たないでください | Rica için ないでください |

## 🧠 Pratik Cümle Örnekleri

```
すみません、もう一度言ってください。
→ Özür dilerim, bir kez daha söyleyin lütfen.

この線より前に出ないでください。
→ Bu çizginin önüne çıkmayın lütfen.

図書館では静かにしてください。
→ Kütüphanede sessiz olun lütfen.

関係者以外、入ってはいけません。
→ Yetkisiz kişiler giremez.

そんな嘘ついちゃいけないよ。
→ Öyle yalan söylenmez.

写真を撮ってもいいですか？
→ Fotoğraf çekebilir miyim?
はい、どうぞ。/ いいえ、撮らないでください。
→ Evet, buyurun. / Hayır, lütfen çekmeyin.
```
