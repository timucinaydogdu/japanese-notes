# 🗂️ İstek / Teklif (意志・依頼表現)

> **Seviye:** N5  
> **Konular:** 〜たい / がほしい / ましょう / ましょうか / ませんか  
> **Son güncelleme:** 2026

---

## Genel Bakış

Bu 5 yapının hepsi **istek veya teklif** bildirir. Ama ton ve hedef farklı:

| Yapı | Anlam | Kime? |
|---|---|---|
| 〜たい | ...istiyorum | Kendi isteğin |
| 〜がほしい | ...istiyorum (nesne) | Kendi isteğin (şey/nesne) |
| 〜ましょう | ...yapalım | Karşılıklı teklif (kesin) |
| 〜ましょうか | ...yapalım mı? | Karşılıklı teklif (soru) |
| 〜ませんか | ...olmaz mı? | Kibarca davet / öneri |

---

## 1. 〜たい — İstiyorum

### Ne yapar?
Konuşmacının **kendi isteğini** ifade eder. Türkçedeki **"...mak istiyorum"** karşılığıdır.

### Yapı
> **ます形 kökü + たい**  
> (ます'ı at, たい ekle)

```
食べます → 食べ + たい = 食べたい   (yemek istiyorum)
行きます → 行き + たい = 行きたい   (gitmek istiyorum)
飲みます → 飲み + たい = 飲みたい   (içmek istiyorum)
します  → し + たい  = したい     (yapmak istiyorum)
```

### Çekim — たい い-sıfat gibi çekilir!

> 💡 **Önemli:** たい eklendikten sonra cümle **い-sıfat gibi** davranır.

| Form | Yapı | Örnek | Türkçe |
|---|---|---|---|
| Olumlu kibar | たい + です | 食べ**たいです** | yemek istiyorum |
| Olumsuz | たい → たく**ない** | 食べ**たくない** | yemek istemiyorum |
| Olumsuz kibar | たくない + です | 食べ**たくないです** | yemek istemiyorum |
| Geçmiş | たい → たかった | 食べ**たかった** | yemek istemedim |
| Geçmiş kibar | たかった + です | 食べ**たかったです** | yemek istiyordum |

```
日本に行きたいです。         → Japonya'ya gitmek istiyorum.
寿司を食べたいです。         → Sushi yemek istiyorum.
今日は仕事をしたくないです。 → Bugün çalışmak istemiyorum。
子供の頃、パイロットになりたかった。
→ Çocukken pilot olmak istiyordum.
```

### たい + がる — Başkasının isteği

> ⚠️ **Önemli kural:** たい sadece **kendi isteğin** için kullanılır. Başkasının isteği için **たがる** kullanılır.

```
私は食べたい。           → Ben yemek istiyorum. ✅
彼は食べたい。           ❌ YANLIŞ (üçüncü kişi için yanlış)
彼は食べたがっている。   ✅ DOĞRU (o yemek istiyor gibi görünüyor)
```

> 💡 **Pratik not:** Sınavda ve günlük dilde üçüncü kişi için たがる oldukça nadir çıkar. Ama kuralı bil.

## 2. 〜がほしい — İstiyorum (Nesne)

### Ne yapar?
Bir **şeye/nesneye sahip olmak istediğini** ifade eder. たい fiil ister, ほしい **isim** ister.

### Yapı
> **[İsim] + が + ほしい**

```
新しいパソコンがほしい。   → Yeni bilgisayar istiyorum.
時間がほしい。             → Zaman istiyorum.
お金がほしい。             → Para istiyorum.
```

### Çekim — ほしい de い-sıfat gibi çekilir!

| Form | Örnek | Türkçe |
|---|---|---|
| Olumlu | 〜がほしい**です** | ...istiyorum |
| Olumsuz | 〜がほしく**ない** | ...istemiyorum |
| Geçmiş | 〜がほし**かった** | ...istiyordum |

```
新しいパソコンがほしいです。       → Yeni bilgisayar istiyorum.
今はお金がほしくないです。         → Şimdi para istemiyorum.
子供の頃、犬がほしかった。         → Çocukken köpek istiyordum.
```

### たい vs ほしい — Fark

> ⚠️ **Bu çift çok karıştırılıyor:**

| | たい | ほしい |
|---|---|---|
| Sonra ne gelir? | **Fiil** (eylem yapmak) | **İsim** (şeye sahip olmak) |
| Örnek | 食べ**たい** (yemek istiyorum) | りんごが**ほしい** (elma istiyorum) |
| Türkçe | "yapmak istiyorum" | "istiyorum / sahip olmak istiyorum" |

```
水を飲みたい。    → Su içmek istiyorum.    (たい: fiil)
水がほしい。      → Su istiyorum.          (ほしい: isim)
```

> 💡 **Akılda kalıcı:** ほしい = "hoş i" → hoşuma giden **şeyi** istiyorum.

## 3. 〜ましょう — Yapalım

### Ne yapar?
Birlikte bir şey yapmayı **teklif veya öneri** olarak sunar. Türkçedeki **"...yapalım"** karşılığıdır. Kararlı, kesin bir teklif tonu var.

### Yapı
> **ます形 kökü + ましょう**

```
食べます → 食べ + ましょう = 食べましょう  (yiyelim)
行きます → 行き + ましょう = 行きましょう  (gidelim)
始めます → 始め + ましょう = 始めましょう  (başlayalım)
```

```
一緒に食べましょう。       → Birlikte yiyelim.
そろそろ帰りましょう。     → Artık gidelim.
日本語で話しましょう。     → Japoncayla konuşalım.
休憩しましょう。           → Mola verelim.
```

> 💡 **Ton:** ましょう kesin teklif. Karşı tarafın onayını beklemeden "hadi yapalım" der gibi.

---

## 4. 〜ましょうか — Yapalım mı?

### Ne yapar?
ましょう'nun **soru hali**. İki farklı kullanımı var:

**Kullanım 1:** Birlikte bir şey yapmayı **soru şeklinde teklif etme**
```
一緒に食べましょうか。     → Birlikte yiyelim mi?
映画を見ましょうか。       → Film izleyelim mi?
```

**Kullanım 2:** Karşındakine **yardım teklif etme**
```
荷物を持ちましょうか。     → Çantanı taşıyayım mı?
窓を開けましょうか。       → Pencereyi açayım mı?
タクシーを呼びましょうか。 → Taksi çağırayım mı?
```

> ⚠️ **ましょう vs ましょうか farkı:**
> ```
> 行きましょう。   → Gidelim. (kesin teklif, sen de gel)
> 行きましょうか。 → Gidelim mi? (soru, onay istiyorum)
> ```

---

## 5. 〜ませんか — ...Olmaz mı? / Davet

### Ne yapar?
Karşındakini bir şeye **kibarca davet etmek** için kullanılır. ましょうか'dan daha **nazik ve kibar** bir tondur. Türkçedeki **"...olmaz mı?, ...ister misiniz?"** karşılığıdır.

### Yapı
> **ます形 kökü + ませんか**

```
食べます → 食べ + ませんか = 食べませんか  (yemez misiniz?)
行きます → 行き + ませんか = 行きませんか  (gitmez misiniz?)
```

```
一緒にランチを食べませんか。   → Birlikte öğle yemeği yemez misiniz?
今度、飲みに行きませんか。     → Bir gün içmeye gitmez miyiz?
日本語を勉強しませんか。       → Japonca çalışmaz mısınız?
```

> 💡 **Ton farkı — üç teklif karşılaştırması:**
>
> | Yapı | Ton | Örnek |
> |---|---|---|
> | ましょう | Kesin, enerjik "hadi yapalım" | 行きましょう！ |
> | ましょうか | Nazik soru "yapalım mı?" | 行きましょうか？ |
> | ませんか | En kibar davet "gitmez misiniz?" | 行きませんか？ |

## ⚠️ Genel Hata Özeti

| Hata | Doğru | Neden? |
|---|---|---|
| 彼は食べたい | 彼は食べたがっている | たい sadece 1. kişi |
| 水をほしい | 水がほしい | ほしい → が ile gelir, を değil |
| 行きたいほしい | 行きたい (fiil) / 新幹線がほしい (isim) | たい fiil ister, ほしい isim |
| 食べたいくない | 食べたくない | たい い-sıfat gibi: たい → たくない |
| 食べましょうですか | 食べましょうか | ましょうか tek parça |

## 🧠 Pratik Cümle Örnekleri

```
日本語がもっと上手になりたいです。
→ Japoncamın daha iyi olmasını istiyorum。

新しいカメラがほしいけど、高いです。
→ Yeni kamera istiyorum ama pahalı.

疲れたね。休憩しましょう。
→ Yoruldun değil mi. Mola verelim.

一緒にプロジェクトを始めましょうか？
→ Birlikte projeye başlayalım mı?

今度、一緒にランチを食べませんか？
→ Bir dahaki sefere birlikte öğle yemeği yemez miyiz?

子供の頃、宇宙飛行士になりたかったです。
→ Çocukken astronot olmak istiyordum.
```
