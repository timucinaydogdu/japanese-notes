# 🗂️ 〜ている (te iru) — Devam Eden Eylem / Durum

> **Seviye:** N5  
> **Konu:** て形 + いる — şimdiki zaman ve durum  
> **Son güncelleme:** 2026

---

## 〜ている nedir?

て形 + いる birleşimidir. Türkçedeki **"-iyor"** şimdiki zamanına karşılık gelir ama sadece bu değil — **sonuçlanmış bir eylemin devam eden durumunu** da ifade eder.

```
食べる   → 食べて + いる = 食べている
書く    → 書いて + いる = 書いている
勉強する → 勉強して + いる = 勉強している
```

> 💡 **Temel:** て形 yap + いる/います ekle. Kibar için います, samimi için いる.

## Çekim Tablosu

| Form | Yapı | Örnek (食べる) | Türkçe |
|---|---|---|---|
| Olumlu kibar | て形 + **います** | 食べ**ています** | yiyorum |
| Olumlu samimi | て形 + **いる** | 食べ**ている** | yiyor |
| Olumsuz kibar | て形 + **いません** | 食べ**ていません** | yemiyorum |
| Olumsuz samimi | て形 + **いない** | 食べ**ていない** | yemiyor |
| Geçmiş kibar | て形 + **いました** | 食べ**ていました** | yiyordum |
| Geçmiş olumsuz | て形 + **いませんでした** | 食べ**ていませんでした** | yemiyordum |

## 2 Farklı Anlam

〜ている'ın **iki farklı kullanımı** var ve bunları ayırt etmek çok önemli:

### Anlam 1: Devam Eden Eylem (Progressive)
Eylem **şu an devam ediyor**. Türkçe "-iyor" ile birebir örtüşür.

```
今、ご飯を食べています。    → Şu an yemek yiyorum.
雨が降っています。          → Yağmur yağıyor.
彼は電話しています。        → O telefonda konuşuyor.
今、コードを書いています。  → Şu an kod yazıyorum.
```

---

### Anlam 2: Sonuçlanmış Eylemin Durumu (Resultant State)
Eylem **geçmişte tamamlandı**, sonucu **hâlâ devam ediyor**.

```
結婚しています。      → Evliyim. (evlenme bitti, evli olma hali devam ediyor)
知っています。        → Biliyorum. (öğrenme bitti, bilme hali devam ediyor)
眼鏡をかけています。  → Gözlük takıyor. (takma bitti, takmış olma devam ediyor)
日本に住んでいます。  → Japonya'da yaşıyorum. (taşınma bitti, orada olma devam ediyor)
```

> ⚠️ **En çok karıştırılan nokta:**
> ```
> 知りますか？      ❌ YANLIŞ (bilmek anlık eylem değil)
> 知っていますか？  ✅ DOĞRU  (bilme hali devam ediyor)
>
> 結婚しますか？    → Evlenecek misiniz? (gelecek)
> 結婚していますか？→ Evli misiniz? (şimdiki durum)
> ```

---

## Anlam 1 vs Anlam 2 — Nasıl Ayırt Ederim?

| Eylem türü | Anlam | Örnek |
|---|---|---|
| Sürekli yapılabilen eylem (yemek, çalışmak, koşmak...) | Anlam 1: Şu an yapıyor | 走っています → Koşuyor |
| Anlık / tek seferlik eylem (evlenmek, giymek, bilmek...) | Anlam 2: O halde bulunuyor | 結婚しています → Evli |

> 💡 **Pratik test:** "Bu eylem saatlerce devam edebilir mi?"
> - Evet → Anlam 1 (devam eden eylem)
> - Hayır, anlık → Anlam 2 (sonuç durumu)

## Konuşma Dilinde Kısaltma

Günlük konuşmada **い düşer**:

```
食べている  → 食べてる   (yiyor)
書いている  → 書いてる   (yazıyor)
勉強している → 勉強してる (çalışıyor)
```

> 💡 Arkadaşlarla てる formu çok doğal. Sınavda ise ている yaz.

## まだ ve もう ile Kullanım

| Kalıp | Anlam | Örnek |
|---|---|---|
| まだ〜ています | Hâlâ yapıyor | まだ食べています → Hâlâ yiyor |
| もう〜ています | Artık / çoktan | もう食べています → Artık yiyor |
| まだ〜ていません | Henüz yapmadı | まだ食べていません → Henüz yemedi |
| もう〜ていません | Artık yapmıyor | もう食べていません → Artık yemiyor |

## ⚠️ Sık Yapılan Hatalar — Özet

| Yanlış | Doğru | Neden? |
|---|---|---|
| 知りますか | 知っていますか | Bilmek = durum, ている gerekir |
| 結婚しますか (şimdi) | 結婚していますか | Evli olma = durum |
| 住みます (şimdi) | 住んでいます | Yaşamak = durum |
| 食べているます | 食べています | います ayrı gelir, ます eklenmez |

## 🧠 Pratik Cümle Örnekleri

```
今、何をしていますか？
→ Şu an ne yapıyorsunuz?

日本語を勉強しています。
→ Japonca çalışıyorum.

彼女はどこに住んでいますか？
→ O nerede yaşıyor?

このバグはもう直っています。
→ Bu bug çoktan düzeldi.

まだ仕事をしています。
→ Hâlâ çalışıyorum.

眼鏡をかけている人は誰ですか？
→ Gözlük takan kişi kim?
```
