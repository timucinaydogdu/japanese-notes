# Kanji Notları

Japonca kanji öğrenimini sistematik olarak takip etmek için oluşturulmuş bir not sistemi.

## Yapı

- **N5** - JLPT N5 (Beginner)
- **N4** - JLPT N4
- **N3** - JLPT N3
- **N2** - JLPT N2
- **N1** - JLPT N1 (Advanced)

## Not Formatı

Her kanji için aşağıdaki bilgiler tutulur:
- **Kanji**: 文字
- **Stroke Sayısı**: Kaç çizgi
- **Radikal**: Temel bileşen
- **Okunuş**: On'yomi ve Kun'yomi
- **Anlamlar**: Temel anlamlar
- **Örnek Kelimeler**: Kullanım örnekleri
- **İpuçları**: Hatırlama tüyoları
- **Öğrenme Tarihi**: İlk öğrenme tarihi

## Örnek Dosya Adlandırması

```
n5/001-日.md
n5/002-月.md
n4/101-学.md
```

Format: `{JLPT_SEVİYESİ}/{SIRA_NUMARASI}-{KANJI}.md`
